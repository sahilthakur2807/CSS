Pseudo Elements

Theory: Pseudo elements style a specific part of an element, such as the first letter, first line, before, or after content.

Usage: They are useful for decorative or structural styling without extra HTML.

Explanation: Pseudo elements help you create added visual content or style part of an element.

Example

.note::before {
  content: "Note: ";
}

.note::after {
  content: " This is added with CSS.";
}

Pseudo elements can add text before or after this sentence.
