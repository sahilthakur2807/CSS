CSS Syntax

Theory: CSS syntax is the format used to write a CSS rule.

Usage: Every rule has a selector and a declaration block containing property-value pairs.

Explanation: The selector chooses the HTML element, and the declaration block defines the styles to apply.

Basic Structure

selector {
  property: value;
}

Example

p {
  color: purple;
  font-size: 20px;
}

This paragraph follows CSS syntax rules.
