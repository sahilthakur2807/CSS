ID Selectors

Theory: An ID selector targets one unique element. It starts with a hash symbol.

Usage: Use IDs for a single important element on a page, such as a header section or a unique box.

Explanation: IDs should be unique in a page, so they are more specific than class selectors.

Example

#special-box {
  background: #fef3c7;
  border: 2px solid #f59e0b;
}
This box uses an ID selector.
