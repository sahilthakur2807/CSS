External CSS

Theory: External CSS is stored in a separate .css file and linked to HTML with a <link> tag.

Usage: It is the best choice for real projects because the same CSS file can style many pages.

Explanation: It separates structure and design, making code easier to maintain and reuse.

Example

<head>
  <link rel="stylesheet" href="style.css">
</head>

/* style.css */
h1 {
  color: #2563eb;
}

This looks like external CSS

In practice, the CSS would live in a separate file.
