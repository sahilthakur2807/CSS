Introduction to CSS

Theory: CSS stands for Cascading Style Sheets. It controls how HTML elements look on the page, including color, spacing, size, layout, and animation.

Usage: Use CSS whenever you want to style HTML content. Without CSS, pages look plain and default.

Explanation: CSS works by selecting HTML elements and applying property-value pairs to them. The browser reads the rules and renders the visual design.

Example

h1 {
  color: steelblue;
  text-align: center;
}

p {
  font-size: 18px;
}
