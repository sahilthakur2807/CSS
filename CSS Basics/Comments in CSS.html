Comments in CSS

Theory: CSS comments are notes written inside the stylesheet that browsers ignore.

Usage: Use comments to explain your code, separate sections, or leave reminders while learning.

Explanation: Comments make CSS easier to read and understand, especially in bigger projects.

Syntax

/* This is a CSS comment */

Example

/* Style the title */
h1 {
  color: navy;
}

/* Style the paragraph */
p {
  font-size: 18px;
}

Commented CSS Example

Comments do not appear on the page, but they help you understand the code.
