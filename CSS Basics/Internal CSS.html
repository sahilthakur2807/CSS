Internal CSS

Theory: Internal CSS is written inside a <style> tag in the HTML file, usually in the <head> section.

Usage: It is useful when one HTML page needs its own styles and you do not want to create a separate CSS file yet.

Explanation: The CSS applies only to that page. It keeps styles organized better than inline CSS.

Example

<head>
  <style>
    h1 { color: orange; }
    p { font-size: 18px; }
  </style>
</head>
