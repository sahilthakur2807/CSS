Box Sizing

Theory: The box-sizing property controls whether width/height include padding and border.

Usage: Use box-sizing: border-box to make element sizing include padding/border (easier layout).

Values

content-box   /* default: width/height apply to content only */
border-box    /* width/height include padding and border */

Demo

content-box (default)

width:200px + padding/border add to size

border-box

width:200px includes padding/border

Tip

Common pattern: *{box-sizing:border-box} to simplify responsive layouts.
