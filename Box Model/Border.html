Border

Theory: Borders surround an element's padding and content; they can be styled with width, style, and color.

Usage: Use borders to separate content, emphasize elements, or create simple UI outlines.

Syntax

element { border: width style color; }

/* Examples */
border: 2px solid #2563eb;
border-top: 4px dashed #f59e0b;
border-radius: 8px; /* rounded corners */
