Types of CSS

Theory: CSS can be written in three main ways: inline CSS, internal CSS, and external CSS.

Usage: Choose the type based on the size of your project and how reusable you want the styles to be.

Explanation: Inline CSS is written inside an HTML element, internal CSS is written inside a <style> tag in the HTML file, and external CSS is kept in a separate .css file.

Main Types

  • Inline CSS: quick for one element
  • Internal CSS: useful for one page
  • External CSS: best for reusable styles across many pages

Example

<p style="color: red;">Inline CSS</p>

<style>
  p { color: blue; }
</style>

/* style.css */
p { color: green; }
