Inline CSS

Theory: Inline CSS is written directly inside an HTML element using the style attribute.

Usage: It is useful for quick testing, single-line changes, or when you need to style just one element.

Explanation: Inline styles have high priority, but they are hard to maintain if used too much.

Example

<h2 style="color: teal; font-size: 28px;">Hello CSS</h2>
