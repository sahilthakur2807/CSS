Grouping Selectors

Theory: Grouping selectors let you apply the same style to multiple selectors by separating them with commas.

Usage: This reduces repetition and keeps the CSS shorter.

Explanation: If several elements need the same style, group them in one rule instead of writing separate rules.

Example

h1, h2, p {
  color: brown;
}
