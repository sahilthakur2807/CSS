Line Height

Concept: Line height controls the vertical spacing between lines of text.

Usage: Proper line height improves readability and makes text easier to scan.

CSS Syntax

element {
  line-height: 1.5;
}

/* Values */
line-height: normal;    /* Default */
line-height: 1;         /* No extra space */
line-height: 1.5;       /* 1.5x font size */
line-height: 2;         /* 2x font size */
line-height: 24px;      /* Fixed pixels */
