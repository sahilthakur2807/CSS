Text Alignment

Concept: Text alignment controls how text is positioned horizontally within its container.

Usage: Use alignment to organize text layout and improve readability.

CSS Syntax

element {
  text-align: left;
}

/* Common values */
text-align: left;      /* Default */
text-align: right;     /* Align to right */
text-align: center;    /* Center text */
text-align: justify;   /* Stretch to fill */

Demo

Left Aligned: This text is aligned to the left side of the container.
Center Aligned: This text is centered in the container.
Right Aligned: This text is aligned to the right side.
Justified: This text is justified, spreading across the entire width of the container to create even edges on both sides.
