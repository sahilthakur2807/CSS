Fonts

Concept: Fonts control how text looks, including the typeface, size, weight, and style.

Usage: Fonts are fundamental to typography and readability. Choose fonts that match your design.

CSS Syntax

element {
  font-family: 'Font Name', sans-serif;
  font-size: 16px;
  font-weight: normal;
  font-style: italic;
}

/* Shorthand */
element {
  font: italic bold 16px Arial, sans-serif;
}
