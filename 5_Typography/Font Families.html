Font Families

Concept: Font families are groups of typefaces. Main categories: serif, sans-serif, monospace, and cursive.

Usage: Choose a font family based on your design needs and readability goals.

CSS Syntax

/* Serif fonts - have decorative lines */
font-family: Georgia, 'Times New Roman', serif;

/* Sans-serif fonts - clean, modern */
font-family: Arial, Helvetica, sans-serif;

/* Monospace - fixed width, coding */
font-family: 'Courier New', monospace;

/* Fallback system */
font-family: 'Font Name', Alternative, generic-family;
