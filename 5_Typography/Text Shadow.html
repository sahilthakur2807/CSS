Text Shadow

Concept: Text shadow adds a shadow effect to text, creating depth and visual interest.

Usage: Use shadows to make text stand out, create depth, or add decorative effects.

CSS Syntax

element {
  text-shadow: offset-x offset-y blur-radius color;
}

/* Examples */
text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3);

/* Multiple shadows */
text-shadow: 
  2px 2px 4px #000,
  -2px -2px 4px #fff;
