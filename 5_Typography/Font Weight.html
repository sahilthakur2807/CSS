Font Weight

Concept: Font weight controls the thickness or boldness of text, from light to extra bold.

Usage: Use different weights to create hierarchy and emphasize important text.

CSS Syntax

element {
  font-weight: normal;  /* 400 */
}

/* Common values */
font-weight: 100;      /* Thin */
font-weight: 300;      /* Light */
font-weight: 400;      /* Normal */
font-weight: 700;      /* Bold */
font-weight: 900;      /* Extra Bold */

/* Keywords */
font-weight: lighter;
font-weight: normal;
font-weight: bold;
font-weight: bolder;
