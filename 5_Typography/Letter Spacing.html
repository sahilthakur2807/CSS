Letter Spacing

Concept: Letter spacing controls the distance between individual characters in text.

Usage: Use it to improve readability, create visual effects, or adjust typography aesthetics.

CSS Syntax

element {
  letter-spacing: 2px;
}

/* Values */
letter-spacing: normal;   /* Default spacing */
letter-spacing: -2px;     /* Tighter spacing */
letter-spacing: 2px;      /* Loose spacing */
letter-spacing: 0.1em;    /* Relative to font size */
