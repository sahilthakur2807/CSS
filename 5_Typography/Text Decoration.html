Text Decoration

Concept: Text decoration adds lines or styling to text, such as underlines, overlines, or strikethrough.

Usage: Use decorations to emphasize text, indicate links, or mark edited content.

CSS Syntax

element {
  text-decoration: underline;
}

/* Values */
text-decoration: none;           /* Remove decoration */
text-decoration: underline;      /* Line below */
text-decoration: overline;       /* Line above */
text-decoration: line-through;   /* Strike through */

/* With color and style */
text-decoration: underline wavy #dc2626;
