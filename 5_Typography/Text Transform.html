Text Transform

Concept: The text-transform property changes the capitalization style of text.

Usage: Use it to automatically convert text to uppercase, lowercase, or capitalize each word.

Explanation: This property changes how text appears without altering the actual text in the HTML file.

CSS Syntax

element {
  text-transform: uppercase;
}

/* Common values */
text-transform: none;
text-transform: uppercase;
text-transform: lowercase;
text-transform: capitalize;
