Background Image

Concept: The background-image property sets an image as the background of an element.

Usage: Use it to add visual depth to headers, sections, or entire pages.

CSS Syntax

element {
  background-image: url('image-path.jpg');
  background-size: cover;
  background-position: center;
}

/* With gradient overlay */
element {
  background: linear-gradient(rgba(0, 0, 0, 0.4), rgba(0, 0, 0, 0.4)), url('image.jpg');
  background-size: cover;
}
