Gradients

Concept: Gradients create smooth transitions between colors. CSS supports linear, radial, and conic gradients.

Usage: Use gradients for backgrounds, overlays, and modern visual effects.

CSS Syntax

/* Linear Gradient */
background: linear-gradient(direction, color1, color2, ...);

/* Example */
background: linear-gradient(135deg, #2563eb, #7c3aed);

/* Radial Gradient */
background: radial-gradient(shape, color1, color2);

/* Conic Gradient */
background: conic-gradient(color1, color2, color3);
