Background Color

Concept: The background-color property sets the background color of an HTML element.

Usage: Use it to add color to elements like boxes, buttons, headers, or the entire page.

CSS Syntax

element {
  background-color: color-value;
}

/* Example */
body {
  background-color: #f8fafc;
}

div {
  background-color: #2563eb;
}
