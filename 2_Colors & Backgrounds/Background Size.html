Background Size

Concept: The background-size property controls how a background image is sized and scaled.

Usage: Use it to make images fit perfectly within elements or tile them.

CSS Syntax

element {
  background-image: url('image.jpg');
  background-size: cover;
  background-position: center;
}

/* Common values */
background-size: cover;      /* Fill entire element */
background-size: contain;    /* Show entire image */
background-size: 100px 100px; /* Fixed size */
background-size: 50% 50%;     /* Percentage */
background-size: auto;        /* Original size */
