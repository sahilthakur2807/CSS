Background Position

Concept: The background-position property sets the starting position of a background image.

Usage: Use it to place images at specific locations within an element.

CSS Syntax

element {
  background-image: url('image.jpg');
  background-position: center;
  background-repeat: no-repeat;
}

/* Common values */
background-position: top left;
background-position: center;
background-position: bottom right;
background-position: 50% 50%;
background-position: 10px 20px;
