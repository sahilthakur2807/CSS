Width and Height

Theory: Width and height control the size of elements. They accept absolute units (px), relative units (%/em/rem), and auto.

Usage: Use explicit sizes for fixed-layout components, and percentage/flex for responsive designs.

Syntax

element { width: 300px; height: 200px; }

/* Relative */
width: 50%;
height: auto;

/* min/max */
min-width: 200px;
max-width: 800px;
