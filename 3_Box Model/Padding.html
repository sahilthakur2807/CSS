Padding

Theory: Padding is the space between an element's content and its border.

Usage: Use padding to create breathing room inside elements (buttons, cards, etc.).

Syntax

/* shorthand */
element { padding: top right bottom left; }

element { padding: 12px; /* all sides */ }
element { padding-left: 8px; }
