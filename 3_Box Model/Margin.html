Margin

Theory: Margin is the space outside an element's border that separates it from other elements.

Usage: Use margins to create space between elements. Margins can be set for each side independently.

Syntax

/* shorthand */
element { margin: top right bottom left; }

/* common */
element { margin: 16px; /* all sides */ }
element { margin-top: 8px; }
