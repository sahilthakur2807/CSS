CSS Box Model

Theory: Every HTML element is a rectangular box composed of content, padding, border, and margin.

Usage: Understanding the box model helps control spacing, sizing, and layout.

Structure

/* Box model layers (inside → outside) */
content
padding
border
margin

Example (visual)

margin (outer)
border (dashed area)
padding
content

Quick tip

Use box-sizing to include padding/border in width/height calculations.
